## Featured Projects

### [ExoBeast](https://github.com/Matt040205/ExoBeast)

Cooperative tower-defense prototype built in Unity 6 with online multiplayer for up to 4 players.

My contribution is focused on the multiplayer and networking layer, including Epic Online Services integration, lobby/session flow, Unity Netcode synchronization, build credential handling, and recurring host/client bug fixes.

**Main technologies:** Unity 6, C#, Netcode for GameObjects, Unity Transport, Epic Online Services, FMOD

### [Ahri V3](https://github.com/Sitr3n01/ahri_agent)

Experimental AI assistant platform built as a modular monorepo with backend, desktop and web packages.

The project explores multi-provider LLM orchestration, persona systems, memory layers, WebSocket streaming, local-first data handling, and shared TypeScript contracts between clients.

**Main technologies:** TypeScript, Python, FastAPI, Electron, React, SQLite, ChromaDB, Turbo, LLM provider integrations

### [Quality Gate](https://github.com/Sitr3n01/quality_review)

Deterministic CI/CD quality gate for AI-assisted codebases.

The project is designed around a simple principle: AI can explain the result, but deterministic checks decide whether a change passes or fails. It uses a baseline/ratchet model to block regressions while allowing legacy projects to improve gradually.

**Main technologies:** JavaScript, Node.js, GitHub Actions, CI/CD, static analysis, agent skill workflows, Claude/Codex integration

### [Privacy Clipboard Sentinel](https://github.com/Sitr3n01/privacy_clipboard_for_android)

Android privacy audit tool that monitors system log events related to clipboard access attempts.

The app is designed for local auditing and does not read clipboard content. It uses logcat-based event detection, local persistence, foreground services, and a reactive Jetpack Compose interface.

**Main technologies:** Kotlin, Jetpack Compose, Room, Coroutines, SharedFlow, Android Foreground Service, ADB/logcat

### [LUMINA](https://github.com/Sitr3n01/apartment_rental_manager)

Local-first Windows desktop application for managing short-term apartment rentals across Airbnb and Booking.com.

The system combines an Electron desktop shell, React frontend and FastAPI backend to synchronize iCal calendars, detect reservation conflicts, generate documents, send notifications and provide a local dashboard.

**Main technologies:** Python, FastAPI, React, Electron, SQLite, SQLAlchemy, Telegram integration, email automation

### [News Portal](https://github.com/Sitr3n01/news_portal)

Multi-site news and institutional portal built with Django.

The project serves a school/institutional website and a public news portal from the same codebase using Django Sites Framework, PostgreSQL, HTMX and Django Unfold.

**Main technologies:** Python, Django, PostgreSQL, HTMX, Alpine.js, Docker Compose, Django Unfold
